# 인사 명령어

사용자에게 시간대에 맞는 개인화된 인사 메시지를 전달한다.

## 시간대별 인사 메시지

현재 시간을 확인하고 다음 기준에 따라 인사한다:

- **06:00 - 11:59**: "좋은 아침입니다, {사용자이름}님! 오늘 하루도 활기차게 시작하세요."
- **12:00 - 17:59**: "좋은 오후입니다, {사용자이름}님! 무엇을 도와드릴까요?"
- **18:00 - 21:59**: "좋은 저녁입니다, {사용자이름}님! 오늘 하루 수고 많으셨습니다."
- **22:00 - 05:59**: "안녕하세요, {사용자이름}님! 늦은 시간까지 열심히 하시네요."

## 사용자 이름 확인

Windows 환경에서는 `USERNAME` 환경변수를, Unix 계열에서는 `USER` 환경변수를 사용한다.
환경변수를 찾을 수 없으면 "사용자"로 대체한다.

## 실행 방법

1. 현재 시간(시)을 확인한다
2. 시간대에 맞는 인사말을 선택한다
3. 사용자 이름을 환경변수에서 가져온다
4. 인사 메시지를 사용자에게 직접 출력한다

## 예시 출력

```
좋은 아침입니다, with28님! 오늘 하루도 활기차게 시작하세요.
```
