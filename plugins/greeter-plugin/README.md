# Greeter Plugin

시간대별 개인화된 인사 메시지를 제공하는 Claude Code 플러그인입니다.

## 기능

- **자동 인사**: 세션 시작 시 자동으로 인사 메시지 표시
- **수동 인사**: `/greet` 명령어로 언제든지 인사 받기
- **시간대별 인사**: 아침/점심/저녁/밤에 따라 다른 인사
- **사용자 이름 포함**: 사용자 이름을 포함한 개인화된 인사

## 설치

```bash
claude --plugin-dir /path/to/greeter-plugin
```

## 사용법

### 자동 인사
Claude Code 세션을 시작하면 자동으로 인사합니다.

### 수동 인사
```
/greet
```

## 시간대별 인사 메시지

| 시간대 | 인사 |
|--------|------|
| 06:00 - 11:59 | 좋은 아침입니다 |
| 12:00 - 17:59 | 좋은 오후입니다 |
| 18:00 - 21:59 | 좋은 저녁입니다 |
| 22:00 - 05:59 | 안녕하세요 (밤) |
